Jubilee Austen Portfolio

This landing page was a concept project for a hypothetical client named Jubilee Austen. In it's early stages, it was a single-page, non-responsive website that helped me sharpen my HTML5 and CSS skills. I used Photoshop design files to create a website that was a pixel-perfect match.Through this project, I learned how to extract elements from PSD files like fonts, spacing, sizing, images, and other design elements.

This was a very basic project that I used to teach myself the fundamentals of coding (folder & file structure, relative links, setting up an efficient work space, finding a work flow that fit me, the very basics of CSS styling, web browser developer tools, proper syntax and best practices, etc.) I ultimately circled back and made this web page mobile friendly. This project kick-started my love for web development.
